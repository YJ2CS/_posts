使用Linux更新 Sakura 最新release

```bash
cd /home/www/htdocs/wp-content/themes

curl -LJO <https://github.com/mashirozx/Sakura/archive/v3.3.9.tar.gz>
```

解压tar.gz文件

```bash
tar -zxvf Sakura-3.3.9.tar.gz
```
